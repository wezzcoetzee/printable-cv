# Product Sense

## What this is

A personal CV for Wesley Coetzee. The primary audience is recruiters and hiring managers
who may view it on the web or print it to A4.

## Success criteria

- Readable at a glance: name, title, and key experience visible without scrolling
- Clean A4 print: sections fit the page, no orphaned headings, no clipped content
- Fast static load: no server round-trips, no blocking resources
- SEO: discoverable by name search, correct structured data, valid sitemap

## Non-goals

- Not a CV builder — there is no editor UI
- No CMS — content is edited directly in `resume-data.tsx`
- No user accounts — there is no authentication
- Not multi-tenant — this is one person's CV, hardcoded
