# Product Specs

## Active specs

None. The site is in content-update mode with no active feature development.

## Completed features

- Single-page static CV with web and print targets
- Command palette (⌘J) with Print action and social links
- JSON-LD structured data for SEO
- OG and Twitter card images
- Cloudflare Pages deployment with PR preview environments
- `llms.txt` and `llms-full.txt` for LLM discoverability

## Adding a spec

1. Create a new markdown file in this directory: `docs/product-specs/[feature-name].md`
2. Include: problem statement, proposed solution, success criteria, and open questions.
3. Add a row to the Active specs table above with a link to the file.
4. Move to Completed when shipped.
